# Configuration

Avant l'usage quotidien, quelques réglages de base structurent Resthome. Ils se
font une fois, puis n'évoluent qu'à la marge.

## Chambres et secteurs

- **Chambres** : créez vos chambres, avec leur **type** (MR, MRS, chambre simple
  ou double…) et leurs **équipements**.
- **Secteurs / unités** : regroupez les chambres par unité de vie.
- Une chambre peut être mise en **maintenance**, **réservée**, ou repassée en
  attribution **automatique** selon l'occupation.

## Tarifs et forfaits

- **Tarifs INAMI** : les montants du **forfait** par **catégorie Katz** — la base
  du remboursement mutuelle.
- **Tarifs d'hébergement** : le prix de la chambre (part résident).
- **Types de suppléments** : le catalogue des prestations facturables (voir [Les
  suppléments](../facturation/supplements.md)).
- **Règles de remise d'absence** et **codes AVIQ** : le paramétrage fin de la
  facturation.

## Mutuelles et données de référence

- **Mutuelles (organismes assureurs)** : la liste utilisée pour l'assurabilité et
  l'eFact.
- **Établissement** : les informations de votre maison (identifiants, coordonnées)
  utilisées dans les documents et les envois eHealth.

## Multi-société

Resthome gère **plusieurs sociétés** dans une même base : chaque société a ses
résidents, ses chambres et sa facturation, cloisonnés. Utile pour un groupe qui
exploite plusieurs établissements.

!!! note "Le journal d'activité"
    Un **journal d'activité** trace les accès aux données sensibles (RGPD). Il est
    consultable depuis la configuration — utile pour la conformité.
