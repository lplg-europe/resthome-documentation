# Premiers pas

Ce guide vous fait découvrir Resthome en quelques minutes : la connexion, les
applications disponibles et le tableau de bord.

## Se connecter

1. Ouvrez votre navigateur à l'adresse fournie par votre établissement
   (par exemple `https://votre-etablissement.resthome.be`).
2. Saisissez votre **adresse e-mail** et votre **mot de passe**.
3. Cliquez sur **Se connecter**.

!!! note "Mot de passe oublié ?"
    Utilisez le lien **« Réinitialiser le mot de passe »** sur la page de
    connexion, ou demandez à votre administrateur.

## Les applications

Resthome est organisé en applications, accessibles depuis le **menu principal**
(l'icône en damier, en haut à gauche) :

| Application | À quoi elle sert |
|---|---|
| **MR/MRS** | Résidents, séjours, facturation, conformité Katz, eHealth. |
| **Soins** | Prescriptions, plans de soins, signes vitaux, notes infirmières. |
| **Repas** | Menus, régimes, suivi nutritionnel, portail familles. |
| **Configuration** | Chambres, tarifs, données de base de l'établissement. |

Selon votre rôle (administratif, soignant, cuisine…), vous ne voyez que les
applications qui vous concernent.

## Le tableau de bord

Le **Tableau de bord** de l'application MR/MRS donne une vue d'ensemble :

- les **tâches en attente** (Katz à faire, MDA à vérifier, lots eFact…) ;
- des **compteurs** cliquables qui ouvrent directement la liste concernée ;
- les **alertes** importantes du mois.

!!! tip "Astuce"
    Cliquez sur un compteur (par exemple « Katz à faire ») pour ouvrir
    directement la liste des éléments à traiter.

## Se repérer dans un écran

- **Menu de gauche / onglets du haut** : les sections de l'application.
- **Fil d'Ariane** (en haut) : votre position ; cliquez dessus pour revenir en
  arrière.
- **Bouton engrenage / menu Actions** : les actions supplémentaires sur la fiche
  ouverte.
- **Chatter** (colonne de droite) : l'historique et les notes d'un enregistrement.
